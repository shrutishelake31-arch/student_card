🎓 Student Profile Card

A modern and responsive Student Profile Card built using HTML5 and CSS3. This project showcases student information in a clean, attractive, and professional card layout while demonstrating core front-end development concepts.

---

✨ Features

- 📱 Fully Responsive Design
- 🎨 Modern and Clean User Interface
- 👤 Student Profile with Image
- 📝 Student Information Section
- 🌈 Beautiful Colors and Typography
- 💡 Smooth Hover Effects
- 🚀 Beginner-Friendly Project

---

🛠️ Technologies Used

- HTML5
- CSS3
