# 🎓 Student Profile Card

A simple and responsive **Student Profile Card** created using **HTML and CSS**. This project displays student information in a clean card layout with styling, borders, colors, and proper alignment.

## 📌 Features

- Student Name Display
- Profile Information Section
- Contact Details
- Clean and Modern Design
- Responsive Layout
- Beginner-Friendly Project

## 🛠️ Technologies Used

- HTML5
- CSS3
